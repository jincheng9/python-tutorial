# (1)和(1,)的类型是什么

## 问题

如下代码

```python
var1 = (1)
type(var1)
var2 = (1,)
type(var2)
var3 = 2,
type(var3)
var4 = 2,3
```

变量var1/var2/var3/var4的类型分别是什么？

## 解答

var1的类型是int

var2/var3/var4的类型都是tuple

* 这是因为Python中用圆括号`()`来创建元组时，如果元组中只有一个元素，必须在这个元素后面添加一个逗号`,`，以区分是定义元组还是仅仅是在括号中使用一个值。如果不加逗号，Python会认为这只是简单的括号表达式，而不是元组。

* 逗号才是Python区分单个元素的值和单元素元组的关键。
